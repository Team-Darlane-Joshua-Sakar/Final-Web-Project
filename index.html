Do you want to be an IT student?

Yes? Then this event will showcase to you the innovation, fun, and teamwork to complete challanges and gain experience of the possibilities of a career in IT.

IT Code Fair is the perfect place for IT students to show off their skills to others or to learn more from others to always be improving on their skills.

WHERE is the IT code fair

Want to come by and check it out? We will be at DoubleTree By Hilton Hotel Esplanade Darwin (116 Esplanade, Darwin City)

IT Code fair offically starts at 9 November, 10 A.M Darwin time.

What is the IT Code Fair?

Code Fair is a yearly event hosted by Charles Darwin University, where students are able to share their IT skills of all experiences, to interact and create digital problem solving programs to help others while connecting students professions in the IT industry.

Participation in these challanges are not necessary but welcomed. Each year this event presents many new challanges to all, creating new ways to innovate the way you think, and the way you code to help solve problems.

WHY it code fair

Amazing challenges and competitions to take part in. Showcase your coding skills to fellow students, teachers and judges. Solve real-life local business issues, win attractive prizes and more. Learn more about the competitions and events taking place.

Coding and Poster Competitions. Events taking place: Employer Speed Dating, Cyber Securtiy Workshop and Academic Excellence Awards
